# Getting Support

Thanks for using OffOn.dev! Here's where to go:

- **Working through a challenge or have a general question?** Start in the
  [OffOn community](https://offon.dev/).
- **Found a bug or have a concrete feature request?** Open an issue in the
  relevant repository using the issue templates.
- **Security issue?** Follow [SECURITY.md](SECURITY.md) — please don't file a
  public issue.

Search existing issues before opening a new one, and remember this is a
volunteer-run open source community — be patient and kind.
