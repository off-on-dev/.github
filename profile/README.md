# OffOn.dev

**OffOn** is a community for open source enthusiasts. We publish hands-on,
recurring challenges for **Cloud Native, OpenTelemetry, AI/ML**, and other open
source skills — plus the docs and tooling around them.

🌐 **[offon.dev](https://offon.dev/)**  ·  🎯 **[Pick a challenge](https://offon.dev/adventures/)**

## Repositories

- **[open-source-challenges](https://github.com/off-on-dev/open-source-challenges)**
  — the challenges, environments, and validation tooling. **Open source** and
  made to be forked: code under MIT, written content under CC BY 4.0.
- **[website](https://github.com/off-on-dev/website)** — the source for offon.dev.
  **Open source**: code under MIT, written content under CC BY 4.0.

The **OffOn** name and logo are reserved — see each repo's `TRADEMARK.md`.

## Contributing

Contributions are welcome — typo fixes included. Read the relevant repo's
`CONTRIBUTING.md` and our
[Code of Conduct](https://github.com/off-on-dev/.github/blob/main/CODE_OF_CONDUCT.md).
We use the [Developer Certificate of Origin](https://developercertificate.org/):
sign your commits with `git commit -s`.
